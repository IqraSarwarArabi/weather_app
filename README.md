## Local Setup
For local setup, you need open weather api key.

- Clone the repositry.
- Make a local .env file in root of your project and set following variables.
```
REACT_APP_OPEN_WEATHER_API_KEY = 
REACT_APP_OPEN_WEATHER_BASE_URL = ""
```
- Run ` npm i `
- Run ` npm start `
